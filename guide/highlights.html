Highlights




  • Drag & Drop


    Define simple and clean properties to set drag & drop for row as well as for column.









  • Sorting


    Use high-performance sorting by one or multiple columns by clicking the headers and fluently change order of sorting.









  • Grouping


    Explore your data in convenient view and group data by specific columns.









  • Editing


    Use predefined and make your own templates to edit cells of various data types.









  • Batch Update


    Update multiple cells in one batch quickly and conveniently.









  • Themes


    Create and switch on custom theme appropriate your system appearance.









  • Row Details


    Provide additional information about rows using their details and define view of the data.









  • Filtering


    Use built-in column filters or implement custom logic and view for them.









  • State Persistence


    Use built-in tools to persist your state settings like filtering, grouping, sorting, etc and point out storage of the state.









  • Condition Builder


    Set powerful filters with AND/OR logic using condition builder.









  • Selection


    Choose necessary mode of selection (row, column, cell or mixed) by setting of only one property.









  • Style API


    Customize your row and column appearance dynamically with our style API.









  • Pinned Columns & Rows


    Pin your data which you want to see constantly to any border.









  • Span Headers & Pivoting


    Organize and extract the significant data using span headers and pivoting feature.









  • Layer System


    Customize the grid view until the data is received.









  • Export/Import


    Save and load your data in popular file formats.
